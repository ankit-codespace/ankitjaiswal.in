Class and Object
Class is basically a blueprint of an object.
Object is a real-world entity containing state and behavior.

Object-Oriented Programming (OOP)

OOP simplifies software design and maintenance by partitioning problems into modular, reusable components.
Let's implement a company class representing our specific state:
class Company:
    boss = "Ankit"
    loc = "Punjab"
    email = "contact@ankitjaiswal.in"
    ob1 = Company()
    print(ob1.boss)
